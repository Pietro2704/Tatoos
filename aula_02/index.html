1.2 Criação de Listas

1.2.1 Listas Ordenadas

São listas que precisam de uma ordem, como por exemplo: colocação dos vencedores de uma corrida

  1. Primeiro Item
  2. Segundo Item
  3. Terceiro Item
  4. Quarto Item

1.2.2 Listas Não Ordenadas

São listas que não precisam de ordem, como por exemplo: lista de supermercado

  • Um item
  • Outro item
  • Mais um item

1.2.3 Listas de Definição

Usada para representar pares de termos e descrição. Ideal quando você quer apresentar uma relação de conceito e explicação.
Usado em um dicionário, cardápio de restaurante e entre outros

HTML
É uma linguagem de marcação de texto usada para estruturar e organizar conteúdos
CSS
É uma linguagem de estilização, usada para estilizar ou personalizar seu HTML
Javascript
É uma linguagem de programação usada para interagir com o usuário que está na sua página web
